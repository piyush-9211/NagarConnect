# AI Training

This directory contains all machine learning resources for the NagarConnect project.

## Folder Structure

datasets/
- raw/         -> Original downloaded datasets
- processed/   -> Cleaned and standardized datasets
- labels/      -> Annotation files (YOLO format)
- train/       -> Training dataset
- valid/       -> Validation dataset
- test/        -> Test dataset

models/
Stores trained model weights and checkpoints.

notebooks/
Contains Jupyter notebooks used for experiments and analysis.

scripts/
Contains preprocessing, training, and evaluation scripts.

experiments/
Stores training logs, metrics, and experiment results.
